# 개요

* 사용자에게 알람이 왔을 때 알려주는 기능



# 설명

* BACK - Spring boot Websocket (STOMP) 사용으로 알람 서버 구축
* FRONT - vue.js 사용 
* web polling 방식으로 실시간 변동 사항을 체크
* 기존 AXIOS를 이용한 HTTP요청을 보내는 방식에서 다소 해더 크기를 줄인 socket통신을 이용
